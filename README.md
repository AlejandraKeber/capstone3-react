# 📖 [Capstone3-react] <a name="about-project"></a>

**[Capstone3-react]** is a mobile web application to check a list of metrics for crypto currency. It is a SPA that fetch currency data from an API and display main information in the home page and more detailed information in the details page.
 The [design guidelines](https://www.behance.net/gallery/31579789/Ballhead-App-%28Free-PSDs%29) are based on Nelson Sakwa Ballhead App

## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

React JS, Redux, JavaScript, HTML and CSS

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need to have installed Node JS

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone https://github.com/AlejandraKeber/capstone3-react.git
```

### Install

Install this project with:

```sh
  cd capstone3-react
  npm install
```

### Usage

To run the project, execute the following command:

```sh
  npm start
```
